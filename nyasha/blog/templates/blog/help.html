Чтобы начать вести микроблог на нашем сайте, достаточно открыть свой любимый Jabber-клиент и добавить в список контактов пользователя {{ settings.JABBER_BOT_SETTINGS.jid }}. Предварительная регистрация на сайте не требуется. Теперь всё, что вы напишете этому пользователю, будет опубликовано в вашем микроблоге.

Псевдоним (ник)

У некоторых пользователей Jabber ID совпадает с почтовым адресом. Если вы не хотите, чтобы ваш адрес отображался на сайте, вы можете выбрать себе псевдоним (ник). Для этого воспользуйтесь командой:
NICK новыйник.

В имени ника разрешается использовать только латинские буквы, цифры и дефис. Ник не может состоять только из цифр. Длина ника — от 2 до 16 символов. Строчные и заглавные буквы различаются, но занять ваш ник в другом написании никто не сможет.

Вы можете изменить ник в любой момент. Ваши подписки и подписки других пользователей на ваш блог остаются в силе. С адреса вида {{ settings.SERVER_DOMAIN }}/jid@domain.com/ на новый адрес {{ settings.SERVER_DOMAIN }}/nick/ устанавливается перенаправление.

Личная информация

Вверху всех микроблогов отображается информация об авторе. Эти данные загружаются автоматически из визитки (vCard).

Статус

Если вы авторизуете контакт Juick-а, то он будет показывать ваш статус в заголовке блога.

{% endcomment %}

Ответы

Для ответа на сообщение #n: наберите «решетку», номер чужого сообщения и, через пробел, текст вашего ответа.
Например: #1234 Спасибо, отличная ссылка!

Для ответа на другой комментарий к сообщению:
#1234/5 Да, я тоже так думаю.

Если вы отвечаете на сообщение другого комментатора (не владельца блога), то имя этого комментатора автоматически добавится впереди вашего сообщения.

Теги

Чтобы добавить тег к сообщению, наберите «звездочку», затем сам тег и, через пробел, текст вашего сообщения.
Например: *тест Пример использования тегов. ;)

Чтобы добавить тег к уже опубликованному посту, воспользуйтесь командой вида: #12345 *имятега. Если такой тег уже был указан, он будет удален.

Рекомендации

Если вам понравился какой-то пост, им можно поделится со всеми своими подписчиками (кроме тех, которые его уже видели) с помощью команды:
! #12345.

Подписка на комментарии

Если вы отправляете ответ на запись из чужого блога, вы автоматически подписываетесь на комментарии к этой записи. Так что, если кто-то ответит на ваш комментарий, вам придет сообщение.

{% comment %}

Выделение текста

В тексте сообщений можно выделять слова и словосочетания *жирным*, /курсивом/ и _подчеркиванием_. Чтобы эти символы преобразовались на вашей странице в нужный шрифт, необходимо, чтобы перед начальным и после конечного символа выделения «*», «/» или «_» стоял пробел или перевод строки.

{%endcomment %}

Ссылки

Если в сообщении вы укажете адрес сайта (URL), то он автоматически преобразуется в ссылку. Чтобы дать ссылку на блог пользователя, укажите его ник или Jabber ID следующим образом: @ник. Ссылку на сообщение можно дать так: #1234.

Адрес, ник и номер сообщения должны быть отделены от предшествующего и последующего текста пробелом или переводом строки.

Команды

Помимо обычных сообщений, вы также можете отсылать специальные команды:

Обратите внимание: команды нужно вводить заглавными буквами!

HELP — показать краткую справку по командам.

NICK новыйник — установить псевдоним (ник).

LOGIN — получить уникальную ссылку для авторизации на сайте.

# — показать последние сообщения из вашей личной ленты (## — вторая страница, ...).

#+ — показать последние 10 сообщений из общей ленты.

@ — показать список популярных персональных блогов.

* — показать список популярных тегов.

#1234 — показать текст сообщения #1234.

#1234+ — показать текст сообщения #1234 и комментарии к нему.

@ник — показать информацию про пользователя. Вместо ника можно указывать Jabber ID пользователя.

@ник+ — показать информацию про пользователя и 10 последних сообщений из его блога. Вместо ника можно указывать Jabber ID пользователя.

*тег — показать 10 последних сообщений с этим тегом.

S — вывести список всех блогов, на которые вы подписаны.

S #1234 — подписаться на комментарии к этому сообщению.

S @ник — подписаться на микроблог этого пользователя. Вместо ника можно указывать Jabber ID пользователя.

U #1234 — отменить подписку на комментарии к этому сообщению.

U @ник — отменить подписку на блог пользователя. Вместо ника можно указывать Jabber ID пользователя.

{% comment %}

ON и OFF — включает и выключает доставку через Jabber новых сообщений, на которые вы подписаны.

ON+ — включает доставку и показывает последние 10 сообщений, которые вы пропустили, пока доставка была выключена.

{% endcomment %}

D #1234 — удалить запись #1234.

D L — удалить вашу последнюю запись.

{% comment %}

BL — показать ваш "чёрный список" пользователей и тегов.

BL @ник — добавить/удалить пользователя из вашего "чёрного списка". Вместо ника можно указывать Jabber ID пользователя.

BL *тег — добавить/удалить тег из вашего "чёрного списка".

PM @ник текст — отослать пользователю личное сообщение.

INVITE user1@example.org user2@example.org — отослать приглашение пользователям (до 10 Jabber ID, через пробел) от вашего имени.

{% endcomment %}

VCARD — обновить информацию о вас из vCard.

PING — проверка связи. :)

{% comment %}

Удаление аккаунта

Когда вы отправляете первое сообщение в Juick, автоматически создается новый аккаунт для вашего Jabber ID. В случае, если аккаунт был создан по ошибке, а вы хотите указать этот Jabber ID в настройках другого аккаунта, отправьте команду DELETEALL. Будьте внимательны, аккаунт удаляется без возможности восстановления.

Личные сообщения

С помощью команды PM вы можете отослать через Jabber личное сообщение пользователю, чей Jabber ID скрыт. Ваши ник и Jabber ID будут указаны в тексте сообщения.
